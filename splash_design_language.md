# Splash UI Design Language

## Purpose

This document defines the visual and structural design language for the
Splash Web UI.

## Layout

-   Sidebar (240px, dark)
-   Top bar (status, weather, time)
-   Main content (card-based grid)

## Typography

-   Font: Inter
-   Sizes: 24px, 18px, 16px, 14px, 12px

## Colors

-   Primary: #1E3A8A
-   Background: #F8FAFC
-   Card: #FFFFFF

## Components

-   Cards
-   Status indicators
-   Charts

## UX Principles

-   Proactive guidance
-   Translate complexity
-   Prioritize attention
